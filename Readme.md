Noninvasive ASP.NET MVC4 Testing
===================

NoninvasiveMVC4Testing contains the examples from my blog post on [Noninvasive Unit Testing in ASP.NET MVC – A Microsoft Fakes Deep Dive] (http://www.richonsoftware.com/post/2012/05/02/Noninvasive-Unit-Testing-in-ASPNET-MVC-A-Microsoft-Fakes-Deep-Dive.aspx) detailing how to noninvasively unit test the default "Internet Application" MVC4 template without modifying the project's code.
